# 🔬 BME Toolkit

Interactive books, RCA tools, and games for **Biomedical Equipment Engineers** — especially those who work with lab equipment.

Built by a Service BME in Qatar 🇶🇦, for the global BME community.

---

## 📚 What’s Inside

| Folder | Title | Type | Description |
|--------|-------|------|-------------|
| `rca-toolkit/` | **RCA Toolkit** | PWA Tool | 7 root‑cause analysis templates (5‑Why, Fishbone, A3, Timeline, C&E Matrix, Checklist, Problem Solving). Installable on any device, works offline. |
| `the-art-of-precision/` | **The Art of Precision** | Interactive Book | Problem‑solving & RCA workbook based on *Problem Solving 101* and *Think Smarter*. |
| `systemquest/` | **SystemQuest: The Lab Detective** | Game | Systems thinking taught through 5 levels of lab equipment failure investigations. Earn badges, print a certificate. |
| `trusted-advisor/` | **From Service Engineer to Trusted Advisor** | Interactive Book | Mindset shift, trust equation, strategic communication for BMEs. |
| `ethics/` | **The Right Choice: Ethics for BMEs** | Interactive Book | Real‑world grey zones, the 4‑question ethical compass, 30‑day habit challenge. |
| `customer-service/` | **The Human Interface: Customer Service for BMEs** | Interactive Book | Phone etiquette, L.A.S.T. complaint handling, F.O.R.D. rapport building, 30‑day mastery plan. |

## 📦 How to Use

1. **Online**: Visit the GitHub Pages link above.
2. **Offline**: Clone this repository and open any folder's `index.html` in a browser. No server required.
3. **Install as an app**: The RCA Toolkit is a PWA. On your phone, tap "Add to Home Screen" to install it like a native app.

### RCA Toolkit (PWA)
- All 7 templates auto‑save to your browser's IndexedDB.
- Version history for every template — view and restore previous snapshots.
- Optional cloud sync to a private GitHub Gist (requires a personal access token with `gist` scope).
- Print any template for professional service reports.
- Customise with your name and logo (they appear on printed documents).

### Interactive Books
- Each book is a self‑contained HTML file with fill‑in exercises.
- Print to A4 for a physical workbook.
- All your entries stay in the page until you refresh (for privacy, nothing is stored).

## 🛠️ Customisation

To add your name/logo to the RCA Toolkit:
- Open `rca-toolkit/index.html` in a browser.
- Type your name in the **👤 Your Name** field (top bar).
- Click **🖼️ Logo** to upload a hospital or personal logo.
- Both will appear on all printed templates.

---

## 📥 Export & Import

- **Export All** (📥): Downloads a JSON backup of all 7 templates.
- **Import** (📤): Restore from a previously exported JSON file.

---

## ☁️ Cloud Sync

1. Create a GitHub personal access token with `gist` scope.
2. In the RCA Toolkit, click **☁️ Sync Gist**.
3. Enter the token. The first sync creates a private gist; subsequent syncs update it.
